# TransFarmaSul - Dashboard Analítico de Logística

Sistema completo de gestão logística e análise acadêmica para empresa de transporte farmacêutico, desenvolvido com React + TypeScript + Tailwind CSS.

## 🎯 Funcionalidades Principais

- **Dashboard executivo** com KPIs, gráficos e análises em tempo real
- **Gestão de frota** com cadastro e monitoramento de veículos
- **Relatórios acadêmicos** com assinatura digital e link privado
- **Dashboard do professor** para avaliação de relatórios
- **Teste Analítico** com geração de dados aleatórios
- **Menu flutuante** com ferramentas e ajuda integrada
- **Mapa interativo** de rotas e operações

## 📦 Stack Tecnológico

- **Frontend**: React 18 + TypeScript
- **Estilização**: Tailwind CSS + shadcn/ui
- **Visualização**: Recharts (gráficos) + Leaflet (mapas)
- **Roteamento**: React Router v6
- **Persistência**: localStorage
- **Testes**: Vitest + Playwright

## 🚀 Como Iniciar

```bash
npm install --legacy-peer-deps
npm run dev
```

Acesse em `http://localhost:5173`
